## ✨ Overview

**IndexedDL.web** is a progressive webapp and landing page for the [IndexedDL](https://github.com/albertolicea00) ecosystem. It loads indexed sites (directory listing servers) on the client side and transforms them into a modern, Netflix‑style interface with thumbnails, categories, grid views, search and filters — all without a backend.

It also hosts the **centralized CSS themes directory** (`themes/`) that the browser extension ([IndexedDL.appx](https://github.com/albertolicea00/IndexedDL.appx)) fetches dynamically, so themes are always up to date across the entire ecosystem.
